Recursos de Acessibilidade

O Sistema de Eleição do Grêmio Estudantil foi projetado para ser acessível a todos os usuários, independentemente de suas habilidades ou limitações.

Recursos Visuais

Alto Contraste

O modo de alto contraste melhora a legibilidade para usuários com baixa visão.

Como ativar: Clique no botão "Alto Contraste" na barra de acessibilidade no topo da página ou use o atalho Alt + C.

Tamanho de Fonte Ajustável

Você pode aumentar o tamanho da fonte para facilitar a leitura.

Como ajustar: Clique nos botões A, A+ ou A++ na barra de acessibilidade ou use o atalho Alt + A para alternar entre os tamanhos.

Suporte a Leitores de Tela

O sistema é totalmente compatível com leitores de tela como NVDA, JAWS e VoiceOver.

  • Todos os elementos interativos possuem rótulos adequados
  • Imagens têm textos alternativos descritivos
  • A estrutura da página segue uma hierarquia lógica de cabeçalhos
  • Elementos dinâmicos são anunciados apropriadamente

Leitura de Página: Clique no botão "Ler Página" na barra de acessibilidade ou use o atalho Alt + R para ouvir o conteúdo da página atual.

Navegação por Teclado

Todo o sistema pode ser utilizado apenas com o teclado, sem necessidade de mouse.

Atalhos de Teclado Globais

Atalho Função
Tab Navegar entre elementos interativos
Shift + Tab Navegar para o elemento anterior
Enter ou Espaço Ativar botões ou links
Alt + C Alternar modo de alto contraste
Alt + A Ajustar tamanho da fonte
Alt + H Ir para a página inicial
Alt + V Ir para a página de votação
Alt + R Ler conteúdo da página

Atalhos na Página de Votação

Atalho Função
1 a 9 Selecionar chapa pelo número

Atalhos na Página de Confirmação de Voto

Atalho Função
Alt + C Cancelar e voltar para a página de votação
Alt + S Marcar a caixa de confirmação e confirmar o voto

Recursos Adicionais

Link para Pular para o Conteúdo

Um link invisível no topo da página permite que usuários de teclado pulem diretamente para o conteúdo principal, evitando a navegação repetitiva pelo cabeçalho.

Como usar: Pressione Tab ao carregar qualquer página para revelar o link "Pular para o conteúdo principal".

Formulários Acessíveis

Todos os formulários do sistema foram projetados para serem acessíveis:

  • Rótulos claros e associados aos campos
  • Instruções explícitas
  • Mensagens de erro descritivas
  • Foco visual aprimorado

Conformidade com Padrões

O Sistema de Eleição do Grêmio Estudantil foi desenvolvido seguindo as diretrizes de acessibilidade WCAG 2.1 (Web Content Accessibility Guidelines) nível AA.

Estamos comprometidos em melhorar continuamente a acessibilidade do sistema. Se você encontrar alguma dificuldade ou tiver sugestões, por favor entre em contato com a equipe de suporte.
